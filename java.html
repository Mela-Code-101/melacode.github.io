What is Java and How Does It Work?

Published on February 8, 2025 by Mela Code 101

Java is a versatile, object-oriented programming language designed for building cross-platform applications, including web services, mobile apps, and enterprise solutions. It's known for its "write once, run anywhere" capability through the use of the Java Virtual Machine (JVM).

How Java Programs Run

  • Java programs are written in .java files and compiled into bytecode using the javac compiler.
  • The bytecode is then run on the JVM, which translates it into machine code specific to the host system.
  • Java applications are typically run with the java command.

History of Java

Java was created by James Gosling and Mike Sheridan at Sun Microsystems in 1991. Initially designed for embedded systems, Java quickly became popular for building platform-independent software. It was released in 1995 and later acquired by Oracle Corporation in 2010. Today, Java powers millions of applications worldwide, from web servers to mobile devices.

Java Loops

Java supports multiple types of loops, including the for, while, and do-while loops, allowing programmers to repeat code execution efficiently.

For Loop Example

for (int i = 1; i <= 5; i++) {
    System.out.println("Loop iteration: " + i);
}

This will print numbers 1 to 5 as the loop iterates.

While Loop Example

int i = 1;
while (i <= 5) {
    System.out.println("Iteration: " + i);
    i++;
}

This example prints iteration numbers until the condition is false.

Slicing in Java

In Java, slicing is primarily applied to arrays or strings. Here's how you can get a substring or subarray:

String Slicing Example

String str = "Hello, World!";
String subStr = str.substring(7, 12); 
// Output: World

This extracts the substring from index 7 to index 12 (excluding 12).

Array Slicing Example

int[] arr = {1, 2, 3, 4, 5};
int[] slicedArr = Arrays.copyOfRange(arr, 1, 4); 

// Output: {2, 3, 4}

This slices the array from index 1 to index 4 (excluding 4).

Java & Hacking

Java plays a significant role in ethical hacking, especially in developing tools and scripts for penetration testing. Some use cases include:

  • Building custom penetration testing tools
  • Writing exploits or payloads
  • Creating network analysis programs
  • Automating hacking tasks in a controlled environment

Many security professionals prefer Java because of its strong typing, which helps catch bugs early, and its vast library ecosystem.
