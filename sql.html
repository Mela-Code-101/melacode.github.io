Step 5: Fix the SQL Bug Challenge

Below is a SQL query with a bug. Try to fix it to make it work correctly. The current query will fail due to an incorrect table name and missing condition in the WHERE clause.

SELECT * FROM users WHERE id = 1;

Bug: There is no table called "users". The query will fail.

Your Task: Update the query to use the correct table name and ensure it returns the expected result. Once you're ready, try it out in your SQL environment.

Fixed SQL Query:

SELECT * FROM employees WHERE
 employee_id = 1;
